# ex05 — 多档位触摸调速呼吸灯

## 功能说明

融合 **触摸引脚** 与 **PWM 呼吸灯**，实现触摸调速：
- LED 持续呼吸渐变
- 每触摸一次，在 1（慢速）→ 2（中速）→ 3（快速）→ 1 之间循环切换

## 硬件

| 引脚 | 功能 |
|------|------|
| GPIO4 | 触摸传感器 T0 |
| GPIO2 | 板载 LED（PWM 输出） |

## 速度档位

| 档位 | 更新间隔 | 效果 |
|:--:|:--:|------|
| 1 | 25ms | 慢速呼吸 |
| 2 | 8ms | 中速呼吸 |
| 3 | 1ms | 快速呼吸 |

## 关键技术

- **`analogWrite()`**：ESP32 PWM 输出，控制 LED 亮度
- **`touchRead()`**：电容触摸检测
- **软件防抖**：80ms 防抖延迟
- **边缘检测**：触摸瞬间触发档位切换

## 代码结构

```cpp
void loop() {
  handleTouch();   // 检测触摸 → 切换档位
  updateBreath();  // 根据档位 → 更新呼吸灯
}
```

模块化设计，触摸检测与呼吸灯逻辑完全分离。
