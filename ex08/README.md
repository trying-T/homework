# ex08 — 物联网安防报警器模拟实验

## 功能说明

融合 **触摸传感器** 与 **Web 服务器**，构建物联网安防报警系统：

- 📱 **网页布防/撤防**：浏览器远程控制
- 🖐️ **触摸触发报警**：布防状态下触摸引脚模拟入侵
- 🚨 **报警锁定**：触发后 LED 高频闪烁，直至撤防
- 🌐 **WiFi 热点**：ESP32 自建 `ESP32-ALARM` 热点

## 硬件

| 引脚 | 功能 |
|------|------|
| GPIO4 | 触摸传感器 T0（入侵检测） |
| GPIO2 | 板载 LED（报警指示灯） |

## 系统状态

| 状态 | 说明 | LED |
|------|------|:--:|
| 未布防 | 触摸无反应 | 灭 |
| 已布防 | 等待触发 | 灭 |
| 报警中 | 触摸已触发 | 80ms 高频闪烁 |

## 网页界面

- 🔵 **布防 Arm** — 进入警戒状态
- 🔴 **撤防 Disarm** — 解除报警，重置状态
- 📊 实时显示系统状态（未布防 / 已布防 / 报警中）

## 网络信息

| 项目 | 值 |
|------|------|
| SSID | `ESP32-ALARM` |
| 密码 | `12345678` |
| IP | `192.168.4.1` |

## 核心技术

- **WiFi AP + WebServer**：独立热点，不依赖路由器
- **`touchRead()`**：电容触摸检测
- **`millis()` 非阻塞闪烁**：报警 LED 高频闪烁不阻塞 Web 服务
- **状态机**：`isArmed` / `isAlarmTriggered` 控制系统行为
- **HTTP 303 重定向**：操作后自动返回主页
