# ex03 — millis() 函数 LED SOS 信号灯

## 功能说明

使用 `millis()` 函数配合**数组驱动状态机**，实现非阻塞式 SOS 摩斯电码信号灯。

## 摩斯电码 SOS

```
S (· · ·) → O (— — —) → S (· · ·)
```

## 状态机设计

通过两个并行数组定义完整的 SOS 闪烁序列：

| 数组 | 作用 | 示例 |
|------|------|------|
| `ledPattern[]` | LED 亮/灭（18 个状态） | `{true, false, ...}` |
| `durationPattern[]` | 每个状态持续时间（ms） | `{200, 200, ...}` |

## 时序参数

| 符号 | 时长 | 含义 |
|------|:--:|------|
| 点 `.` | 200ms 亮 + 200ms 灭 | 短闪 |
| 划 `-` | 600ms 亮 + 200ms 灭 | 长闪 |
| 字母间隔 | 600ms | S 与 O 之间 |
| SOS 间隔 | 2000ms | 完整 SOS 之间 |

## 关键知识点

- **状态机模式**：用数组描述完整行为序列，避免复杂 if-else
- **`millis()` 非阻塞**：不阻塞其他任务
- **自动循环**：`patternIndex >= patternLength` 时重置为 0
